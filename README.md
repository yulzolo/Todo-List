# Todo List (C++)

## Функционал

- add <название> — добавить задачу
- list — вывести список задач
- done <номер> — отметить задачу выполненной
- undone <номер> — вернуть задачу в статус «не выполнена»
- delete <номер> — удалить задачу
- clear — удалить все выполненные задачи

## Формат файла

Каждая строка файла имеет вид:

status|task_title

Пример:

```
0|Buy milk
1|Learn C++
0|Finish project
```

0 — задача не выполнена
1 — задача выполнена
