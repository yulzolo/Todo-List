# Todo List (C++)

Консольное приложение для управления списком задач.

## Реализованный функционал

* add <название> — добавить задачу
* list — вывести список задач
* done <номер> — отметить задачу выполненной
* undone <номер> — вернуть задачу в статус «не выполнена»
* delete <номер> — удалить задачу
* clear — удалить все выполненные задачи
* exit — завершить программу

## Формат хранения данных

Каждая строка файла имеет вид:

status|task_title

Пример:

```
0|Buy milk
1|Learn C++
```

Где:

```
* 0 — задача не выполнена
* 1 — задача выполнена
```

## Обработка ошибок

Программа обрабатывает:

* отсутствие файла данных;
* пустой ввод;
* некорректный номер задачи;
* попытку обращения к несуществующей задаче.

## Сборка

g++ src/main.cpp -o todo

## Запуск

./todo
